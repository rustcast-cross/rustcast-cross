# Rustcast - an alternative to Raycast thats open source and in rust

> Raycast & Rustcast are productivity tools. They provide a popup search bar
> that people can use to do many things, like opening apps, calculators,
> quick-notes, etc.

![RustCast Demo PreRelease V1](docs/rustcast_v1_preview.png)

## Installation:

1. Go to the
   [releases page](https://github.com/unsecretised/rustcast/releases/latest)
2. Download the `.zip` file
3. Open it, and if there is a "this app is damaged", run the command
   `xattr -cr <RustCast.app directory here>`
